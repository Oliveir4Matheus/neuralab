## 1. Melhora no processo de distribuição das pendências do ponto
-
	Atualmente ficamos na dependência da Itos para receber o relatório de ocorrências de ponto o que causa um certo atraso nas tratativas de ponto, um acumulo nas pendências e como consequência uma falta de visibilidade da realidade dos fatos apresentados no ponto como faltas, horas extras, débito e por ai vai.

## 1.1. Processo ETLD
-
	Para resolver essa questão eu estruturei um processo que chamo pelo acrônimo ETLD que visa a extração, tratamento, carregamento e distribuição das pendências do ponto.
## 1.2. Um pouco sobre cada etapa
-
	* Extração: Subprocesso que consiste na extração ( duas vezes ao dia ) das pendências do ponto de todas as bases
	-
	* Tratamento: Subprocesso que consiste no tratamento dos dados extraídos da Sênior
	-
	* Carregamento: Subprocesso que consiste no salvamento dos dados extraídos, já tratados ,para uma planilha de Excel, o envio dos dados para o meu e-mail @swissport e  o salvamento dos dados recebidos dentro do Sharepoint de inconsistências com a nomeação seguindo o seguinte formato: YYYYMMDD_inconsistencias.
	-
	* Distribuição: Subprocesso que consiste na distribuição da última planilha salva dentro do sharepoint de inconsistência para todos os time keepers via E-mail.
## 1.3. Como vai funcionar cada etapa
-
	* Extração, Transformação e Carregamento: Esses subprocessos será realizado por meio de um painel desenvolvido por mim já configurado para realizar a extração dos dados todos os dias às 07:30 e às 16:00 e após a extração um conjunto de scripts vão realizar o tratamento dos dados, o salvamento deles em uma planilha de excel e o envio desses dados para o meu e-mail, onde, assim que receber os dados irá acionar um fluxo do power automate responsavél por salvar os dados recebidos de um destinatário X no Sharepoint já no formato de nomeação correto .
	-
	* Distribuição: Esse subprocesso será realizado através de um fluxo do power automate, onde, sempre que uma planilha de Excel for salva no Sharepoint de inconsistência, um e-mail será disparado para todos os time keepers com esse arquivo anexado.
